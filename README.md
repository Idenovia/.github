# Qu'est-ce qu'Idenovia ?

**Idenovia** est une petite société strasbourgeoise de développement des services orientées vers le télésecrétariat. Fondée en 2011, elle a développé et maintient encore à ce jour les logiciels *Serenovia*, *Wialo* et *Calendovia*. Les deux services principaux, c'est-à-dire Serenovia et Calendovia, ont été sauvegardés en local via le SVN avant leur migration sur GitHub mi-2022.

# Pourquoi le code n'est-il pas accessible hors de l'organisation ?

Le code d'Idenovia est actuellement en closed-source, ce qui veut dire que le code n'est accessible qu'aux membres de l'organisation. Cela est éventuellement justifié par le fait que les logiciels sont payants de base. Mais cela n'empêchera pas le fait que quelques uns des composants seront peut-être bientôt open-source.
